# homebrew-tap

Homebrew tap for [comux](https://github.com/adulwitkku/comux) — a local-first AI orchestrator
that runs coding agents visibly inside [cmux](https://cmux.com).

```bash
brew install adulwitkku/tap/comux
```

This is equivalent to:

```bash
brew tap adulwitkku/tap
brew install comux
```

comux also needs a running cmux, Ollama serving `gemma4:12b-mlx`, and an Agent CLI (`pi`) on
your `PATH`. See the [comux README](https://github.com/adulwitkku/comux#requirements).
